## Функции сортировок

1. **Пузырьковая сортировка** (`bubble_sort`)
   - Основано на: многократном проходе и обмене соседних элементов
   - Скорость сортировки: O(n²) — медленная, но простая

2. **Быстрая сортировка** (`quick_sort`)
   - Основано на: разбиении вокруг опорного элемента (pivot)
   - Скорость сортировки: O(n log n) в среднем, O(n²) в худшем

3. **Сортировка подсчётом** (`counting_sort`)
   - Основано на: подсчёте количества каждого значения
   - Скорость сортировки: O(n + k), где k — диапазон значений

4. **Поразрядная сортировка** (`radix_sort`)
   - Основано на: сортировке по разрядам (единицы, десятки и т.д.)
   - Скорость сортировки: O(d·(n + base)), где d — число разрядов

5. **Блочная сортировка** (`bucket_sort`)
   - Основано на: распределении по "корзинам", затем сортировка каждой
   - Скорость сортировки: O(n) в среднем при равномерном распределении

6. **Пирамидальная сортировка** (`heap_sort`)
   - Основано на: построении и извлечении из двоичной кучи (max-heap)
   - Скорость сортировки: O(n log n) гарантированно

## Факториал и числа Фибоначчи

### Вычисление факториала
1. Итеративно → `factorial(n)`
2. Рекурсивно → `factorial_recursive(n)`

### Вычисление чисел Фибоначчи
1. Итеративно (через список) → `fibo(n)`
2. Рекурсивно → `fibo_recursive(n)`

## Структуры данных

- **Стек** (`Stack`): поддерживает `push`, `pop`, `peek`, `min()` за O(1)
- **Очередь** (`Queue`): поддерживает `enqueue`, `dequeue`, `front`

Все структуры выбрасывают `IndexError` при операциях над пустой структурой.
